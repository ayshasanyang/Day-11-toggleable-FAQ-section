Frequently Asked Questions

What is this project about? ▼
This is a simple FAQ section where users can toggle answers with smooth animations.
How does the animation work? ▼
The answer expands and collapses using CSS transitions and JavaScript.
Can I customize this? ▼
Yes! You can change the colors, font sizes, and add more FAQs as needed.
Is this mobile-friendly? ▼
Yes! The FAQ section is fully responsive and works on all screen sizes.
How can I add more FAQs? ▼
Simply copy and paste a new `.faq-item` block in the HTML and change the text.
